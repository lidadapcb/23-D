# Afl_FFT.c 编码问题修复指南

## 🔧 问题说明

Afl_FFT.c 文件中存在大量的**中文注释编码错误**（乱码），这是因为：
- 原文件使用 **GBK 编码**（中文 Windows 默认）
- 显示环境使用 **UTF-8 编码**
- 编码不匹配导致乱码显示

## 🔄 编码修复方案

### 方案 1：自动修复（推荐）
如果您使用 VS Code，可以：
1. 打开文件 → 右下角选择编码
2. 选择 **"GBK"** 打开文件
3. 选择 **"Save with Encoding"** → **"UTF-8"** 保存
4. 这样可以保留原始注释（虽然乱码）但能正确显示

### 方案 2：手动替换乱码
使用查找替换功能，将所有 `/* 乱码注释 */` 替换为上表中的含义对应的正确中文注释。

### 方案 3：使用本文档作参考
虽然代码中有乱码，但可以参考本文档了解每个注释的含义，不影响代码功能。
